Design Decisions
1

EV/PHEV Preset Dropdown

Grouped by Tesla, Other EVs, and PHEVs. Selecting a preset auto-fills battery kWh — identical pattern to the existing TankPresets component. Garage vehicles with a saved kWh field will also appear here.

2

Dual-Slider Charge Range

"Current %" and "Target %" sliders replace the single fuel-level gauge. Matches how EV drivers naturally think: "I'm at 35%, I want to get to 80%." Number boxes show live values above sliders.

3

Electricity Rate + Auto-Lookup

Calls the same EIA API used for gas prices but hits the electricity retail endpoint. Shows state average as a hint below the field. User can override manually.

4

Efficiency Field (Optional)

If the user enters mi/kWh, the results card shows "Range Added." If skipped, only kWh needed + cost are shown. Keeps the form lightweight for users who just want cost.

5

Charging Time Estimates

Three tiers: Level 1 (1.4 kW), Level 2 (7.2 kW default), and DC Fast (50 kW). Calculated from kWh needed ÷ charger power. Level 2 kW is user-adjustable in a future iteration.

6

Gasoline Comparison (Desktop)

Desktop results card includes an inline comparison: same distance in a 30 MPG gasoline car at current local gas price. Quantifies EV savings in dollars — great for social sharing.

7

PHEV Smart Banner

When a PHEV preset is selected (RAV4 Prime, Volt, etc.), a banner appears prompting the user to use Target Fill for gasoline cost. Prevents confusion about partial EV range.

8

Teal Accent = Electric

All EV-specific UI uses the existing brand teal (#1EB68F / #005F4A) instead of amber. This creates an instant visual language: amber = gas, teal = electric. No new colors needed.

Exact Build Roadmap
Phase 1

Math Engine

  • lib/calculations.ts — add calcEvCharge() and calcEvBudget()
  • Inputs: batteryKwh, currentPct, targetPct, pricePerKwh, efficiencyMiKwh?
  • Outputs: kWhNeeded, cost, rangeAdded?, chargeTimesObj
  • Pure functions, no side effects — same pattern as calcTargetFill()
⏱ ~1 hour
Phase 2

EV Presets

  • lib/evPresets.ts — new file with typed preset array
  • Tesla (3, Y, S, X, Cybertruck), Bolt, Leaf, Mach-E, ID.4, Ioniq 6, EV6, Rivian R1T/R1S, BMW i4
  • PHEV section: RAV4 Prime, Prius Prime, Volt, Escape PHEV, Wrangler 4xe
  • Each entry: label, batteryKwh, defaultEfficiency, isPHEV boolean
⏱ ~30 min
Phase 3

Electricity Rate API

  • app/api/electricity-price/route.ts — new route
  • EIA v2 electricity endpoint: api.eia.gov/v2/electricity/retail-sales/data/
  • Same EIA_API_KEY env var (no new keys needed)
  • Returns state avg + US avg $/kWh
  • Caches result for 24 hrs to stay under EIA rate limits
⏱ ~1 hour
Phase 4

EV Form Component

  • components/EvCalculatorForm.tsx — new component
  • Vehicle preset dropdown (EV + PHEV groups)
  • Battery kWh input with preset auto-fill
  • Dual charge sliders: currentPct + targetPct
  • Electricity rate input + 📍 lookup button
  • Efficiency field (optional)
  • PHEV banner (conditional on isPHEV preset)
⏱ ~3 hours
Phase 5

Results Display

  • Results card inside EvCalculatorForm
  • kWh needed, cost, range added (if efficiency provided)
  • Animated charge bar (existing → target fill)
  • Charging time grid: L1 / L2 / DC Fast
  • Desktop: gasoline comparison footnote
  • Results use teal palette (not amber) for EV branding
⏱ ~2 hours
Phase 6

Tab Integration

  • components/CalculatorTabs.tsx — add 'ev' to CalcTab type
  • Add ⚡ EV tab pill between Budget and existing tabs
  • Conditionally render EvCalculatorForm when tab === 'ev'
  • Persist selected tab to localStorage (existing pattern)
  • EV tab uses teal active state instead of amber
⏱ ~1 hour
Phase 7

Translations

  • lib/translations.ts — new ev block
  • All label strings in EN + ES
  • Spanish: "Tasa de electricidad", "Carga actual", "Carga objetivo", etc.
  • Follow existing t.xxx pattern used in TankPresets, TargetFillForm
⏱ ~30 min
Phase 8

Help Page + Memory

  • app/help/page.tsx — add EV Calculator FAQ section
  • Standing rule: audit help page after every major feature
  • MEMORY.md — add EV tab to completed features
  • Smoke test: mobile + desktop, preset auto-fill, rate lookup, PHEV banner
⏱ ~30 min
