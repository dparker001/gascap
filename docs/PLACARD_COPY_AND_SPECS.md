## Placard 1: Pump Placard (4" × 6" laminated)

### Design Layout

```
┌────────────────────────────┐
│  [HEADLINE VARIANT — see   │  ← Large bold white text on dark green bg
│   A/B/C options below]     │    Font size: ~28–32pt
│                            │
│  [QR CODE — centered]      │  ← Minimum 1.5" × 1.5" for pump distance scan
│  gascap.app                │  ← Small white text below QR
│                            │
│  ✦ Free — no app needed    │  ← 3 benefit bullets, teal bullet points
│  ✦ Live local gas prices   │    Font size: ~11pt white
│  ✦ Know before you fill up │
│                            │
│  [GasCap™ logo — small]    │  ← Bottom center, white
└────────────────────────────┘
```

**Back of card (small text):**
```
Placard Code: [XXXXXXX]     admin@gascap.app
```

### Specs
- **Size:** 4" × 6" (portrait)
- **Bleed:** 0.125" on all sides
- **Resolution:** 300 DPI minimum
- **Color mode:** CMYK
- **Finish:** Matte lamination (weatherproof — pump placards face outdoor elements)
- **Stock:** 14pt card stock + gloss or matte laminate
- **Attachments:** 4× neodymium disc magnets (⅝" or 1"), adhered to back corners with 3M adhesive
- **Magnet product:** DIYMAG or FINDMAG ⅝" adhesive-backed disc magnets (Amazon ~$8/50-pack)

### Headline Variants (A/B/C — use admin dashboard to track performance)

| Variant | Headline |
|---|---|
| A | **Know Before You Fill Up** |
| B | **Don't Guess at the Pump** |
| C | **Stretch Your Gas Budget** |

Print equal quantities of each variant. Track via separate QR codes in admin dashboard.

```
└──────────────────┘
```

### Specs
- **Size:** 4" × 4" (square) or 4" × 6" (portrait)
- **Bleed:** 0.125" on all sides
- **Resolution:** 300 DPI minimum
- **Color mode:** CMYK
- **Material:** Clear or white static cling vinyl
- **Print side:** Face (non-cling side) — applied to interior of window so design faces OUT
- **Attachment:** Press firmly onto clean, smooth glass. No adhesive. Removable.
- **Application tip:** Clean glass with isopropyl alcohol before placing. Smooth out bubbles from center outward.

## QR Code Specifications

Each placard carries a unique QR code tied to its placement code.

**QR URL format:**
- English: `https://gascap.app/q/[CODE]` — e.g. `https://gascap.app/q/ORL003C`
- Spanish (bilingual): `https://gascap.app/q/[CODE]?lang=es`

**QR code sizing:**
| Placard Type | Minimum QR Size | Recommended QR Size |
|---|---|---|
| Pump Placard | 1.5" × 1.5" | 2" × 2" |
| Tent Card | 1.25" × 1.25" | 1.75" × 1.75" |
| Window Cling | 2.5" × 2.5" | 3" × 3" |

**QR code styling:**
- Use square QR modules (not rounded) for reliability at distance
- Add GasCap™ logo to QR center (up to 25% of QR area — does not affect scannability)
- Add teal (#1EB68F) QR border / margin (0.25")
- Dark modules: #1E2D4A (navy) or #005F4A (dark green) on white background

**QR code generator:** Use qr-code-generator.com or Canva's QR generator.
For bulk generation (10+ codes), use a QR batch generator or API.

**Test every QR code** before printing — scan with both iOS and Android, from the expected scan distance.

## Placement Code Naming Convention

**Format:** `[CITY3][NUM3][TYPE1]`

| Part | Length | Example | Meaning |
|---|---|---|---|
| CITY3 | 3 chars | `ORL` | Orlando |
| NUM3 | 3 digits | `003` | 3rd location in territory |
| TYPE1 | 1 char | `C` | Counter |

**Type codes:**
- `P` = Pump (exterior)
- `C` = Counter / Register
- `W` = Window (cling)
- `F` = Flyer / Bulletin Board

**City codes (starter list):**

## Pre-Print Checklist

Before sending files to printer:

- [ ] QR codes generated and tested (scan on iOS + Android)
- [ ] Placard codes printed on card backs in small text (8pt minimum)
- [ ] Correct headline variant assigned to each batch (A / B / C)
- [ ] Bleed + safe zone set per spec
- [ ] Color mode set to CMYK (not RGB)
- [ ] Resolution 300 DPI minimum
- [ ] Magnet adhesive layer included in pump placard order
- [ ] Admin dashboard codes pre-created before print run
